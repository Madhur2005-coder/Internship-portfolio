# Student Performance Predictor

## Overview

A machine learning web application that predicts a student's expected exam performance based on academic habits and engagement metrics.

## Features

* Predict exam score
* Interactive sliders for input
* Attendance analysis
* Study time evaluation
* Assignment completion tracking
* Personalized recommendations

## Technologies Used

* Python
* Machine Learning
* Pandas
* Scikit-Learn
* Web Framework (Flask/Django/Streamlit)

## Inputs

* Daily Study Time
* Attendance
* Sleep Hours
* Class Participation
* Assignment Completion
* Previous Exam Score

## Output

* Predicted Score
* Readiness Assessment
* Performance Factors
* Improvement Suggestions
